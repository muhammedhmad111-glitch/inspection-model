# CPIIS — نظام إدارة تفتيش مصنع الأسمنت

نظام متكامل لإدارة أعمال التفتيش والصيانة في مصنع **CIMPOR AMREYAH** — واجهة عربية (RTL).

🌐 **النسخة المباشرة:** https://inspection-model.vercel.app

## المميزات

- **البيانات الأساسية** — هيكل الأصول: المناطق ← الأقسام ← المعدات ← أجزاء المعدات.
- **مكتبة التفتيش + محرّك جدولة دوري** — يولّد مهام التفتيش تلقائيًا حسب التكرار
  (يومي/أسبوعي/شهري/ربع سنوي...) ويوزّعها على أيام العمل (الأحد→الخميس).
- **تنفيذ الفحص** — واجهة جاهزة للموبايل مع قوائم تحقّق وتقييم حالة المعدة والمرفقات.
- **الملاحظات وإجراءات الصيانة** — تسجيل، تصعيد، ومتابعة.
- **الإشعارات وسجل التدقيق** (Audit Trail).
- **لوحات المؤشرات (KPIs) + عرض المعدة 360 + التقويم + التقارير**.
- **الأدوار والصلاحيات (RBAC)** — مصفوفة صلاحيات يديرها المدير العام، مع تسجيل ذاتي
  وموافقة.

## التقنيات

- **الواجهة:** Next.js 16 (App Router, Turbopack) · React 19 · TypeScript · Tailwind CSS ·
  shadcn/ui — داخل مجلد [`web/`](web/).
- **الخلفية:** Supabase — Postgres + RLS + Auth + Storage + Edge Functions.

## التشغيل محليًا

```bash
cd web
npm install
# أنشئ web/.env.local وضع فيه:
#   NEXT_PUBLIC_SUPABASE_URL=...
#   NEXT_PUBLIC_SUPABASE_ANON_KEY=...
npm run dev
```
